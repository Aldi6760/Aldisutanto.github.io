About Me

Halo Perkenalkan Nama Saya Aldi sutanto Saya Mahasiswa universitas budi luhur fakultas teknologi informasi prodi teknik informatika peminatan saya di prodi teknik informatika adalah programming expert,Saya memiliki ketertarikan besar dalam pengembangan perangkat lunak dan senang belajar serta mengeksplorasi hal-hal baru di bidang ini,selama studi, saya telah mengembangkan berbagai keterampilan teknis seperti pemograman "html,css,javascript" Saya juga telah mengerjakan beberapa proyek, baik secara individu maupun dalam tim, yang dapat Anda lihat di portofolio ini. Melalui proyek-proyek ini, saya belajar untuk memahami kebutuhan pengguna, bekerja dengan tenggat waktu, dan menyelesaikan tantangan-tantangan kreatif maupun teknis.

Sebagai fresh graduate, saya memiliki semangat yang tinggi untuk belajar dan tumbuh dalam lingkungan profesional. Saya percaya bahwa setiap proyek adalah kesempatan untuk terus berkembang dan saya sangat bersemangat untuk berkontribusi dengan ide-ide segar dan energi positif. Jangan ragu untuk menghubungi saya jika Anda tertarik untuk bekerja sama atau berdiskusi lebih lanjut. Saya sangat terbuka untuk kesempatan baru dan siap untuk berkontribusi di bidang pengembangan perangkat lunak
